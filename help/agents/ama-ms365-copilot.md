# Adobe Marketing Agent for [!DNL Microsoft 365 Copilot]

Adobe Marketing Agent for [!DNL Microsoft 365 Copilot] is an AI agent that you can use to retrieve Adobe Experience Platform insights directly from [!DNL Microsoft 365 Copilot] surfaces such as [!DNL Teams] and [!DNL Copilot Chat]. Essentially, Adobe Marketing Agent for [!DNL Microsoft 365 Copilot] embeds [Experience Platform AI Assistant](../ai-assistant/ai-assistant-ui.md) and [Agent Orchestrator](agent-orchestrator.md) inside [!DNL Microsoft 365] so that you can ask questions and retrieve marketing insights without leaving [!DNL Microsoft 365].
