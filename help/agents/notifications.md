# Notifications Agent

The Notifications Agent adds a conversational AI layer within [AI Assistant](../ai-assistant/ai-assistant-ui.md) that you can use to summarize, interpret, and act on your notifications using natural language, thereby turning a passive feed into an active assistant. Instead of scanning a long list, you can go from "I have 47 notifications" to "2 need my approval by Friday, 3 datasets have conflicts, the rest is FYI" in a single exchange.

## What Notifications Agent can do

Notifications Agent supports the following capabilities:

| Capabilities | Description |
| --- | --- |
| **Notification summaries and insights** | Get concise overviews of your notifications, clearly distinguish between those that require action and those that are just informational, and understand both the purpose of a notification and the reason you received it. |
| **Operational actions** | Take action on notifications directly through conversation—mark them as read, snooze them, or set reminders for individual or grouped notifications. |
| **Preferences and settings** | Enable or disable notification channels (email, Slack, in-app) per solution and category, without navigating Settings. |
| **Navigation and deep linking** | Jump from a notification directly to the relevant product area via deep links. |
| **Collaboration and sharing** | Share notifications or send announcements to specific users or the entire organization (authorized users) |

## Access Notifications Agent

You can access Notifications Agent in the following ways:

- **From AI Assistant**: Open AI Assistant and enter any notification-related prompt.
- **From the prompt library**: Select a suggested prompt that fits your requirement.

## Sample prompts

Read the following sections for examples of prompts you can use with Notifications Agent.

### Summaries and insights

- "Summarize my recent notifications."
- "Which of my notifications need my action versus what's just FYI?"

### Actions

- "Mark all non-actionable notifications as read."
- "Remind me about the APAC Winback Campaign notification in an hour."

### Preferences

- "Enable Slack notifications for Journey Optimizer."
- "Switch my daily digest to weekly."

### Collaboration

- "Share this notification with {EMAIL_ADDRESS_OF_USER_IN_YOUR_ORG}."

## Best practices

Keep the following tips in mind when using Notifications Agent:

- **Be specific** — Clearly name the app, category, or notification so that the agent can act on the intended target.
- **Chain prompts** — You can ask for a summary, follow up by focusing on a specific notification, and then take an action, all within a single conversation.
- **Answer clarifying questions** — When the agent asks for confirmation about the app, channel, or notification, be sure to respond so that changes are applied correctly.
