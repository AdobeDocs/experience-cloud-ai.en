# Data Validation Agent

The Data Validation Agent is an agent that you can use to perform statistical and semantical validations on existing Adobe Experience Platform data, or new data that you want to ingest to Experience Platform. You can use the Data Validation Agent to fill essential gaps in both the onboarding process and the day-to-day diagnostics for existing Experience Platform data. With the Data Validation Agent, you can reduce manual QA and accelerate time-to-value for your customers.
